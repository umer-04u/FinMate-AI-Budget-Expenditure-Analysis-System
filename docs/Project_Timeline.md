# Project Timeline (Gantt Chart)

This timeline assumes a 12-14 week schedule for the Final Year Project.

## Phase 1: Setup & Research (Weeks 1-2)
- [x] **Week 1**: Define Goals, Features, Requirements Gathering.
- [ ] **Week 2**: Tech Stack Selection, Project Init, Competitor Research.

## Phase 2: Data Collection & Preprocessing (Weeks 3-5)
- [ ] **Week 3**: Dataset sourcing (Kaggle), Synthetic data generation.
- [ ] **Week 4**: Data Cleaning, EDA, Handling nulls/outliers.
- [ ] **Week 5**: Feature Engineering (Lag features, Rolling averages, Categorization labeling).

## Phase 3: Model Design & Development (Weeks 6-9)
- [ ] **Week 6**: Anomaly Detection Model (Isolation Forest).
- [ ] **Week 7**: Spending Prediction Model (Regression/Time-Series).
- [ ] **Week 8**: Budget Recommendation Logic & Categorization NLP.
- [ ] **Week 9**: Model Training, Hyperparameter Tuning, Evaluation & Saving Models.

## Phase 4: Application Development (Weeks 10-13)
- [ ] **Week 10**: Backend API Setup (FastAPI), Database Design.
- [ ] **Week 11**: Frontend UI Setup (React), Dashboard Design.
- [ ] **Week 12**: Integration (Connecting Frontend to ML APIs).
- [ ] **Week 13**: Chatbot Implementation & "Wow" Factor UI Polish.

## Phase 5: Testing & Final Adjustments (Weeks 14-16)
- [ ] **Week 14**: Functional Testing, Bug Fixing.
- [ ] **Week 15**: Performance Optimization, Documentation.
- [ ] **Week 16**: Final Report Gen, Presentation Prep.

## Responsibility Mapping
- **Student**: Full Stack Dev, ML Engineering, Documentation.
- **Supervisor**: Code Review, Scope Management.
